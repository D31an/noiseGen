NoiseGen Example

The background noise in this page is dynamically generated through JavaScript (unless your browser doesn't support the canvas element, in which case a fallback image is used).

Options:
  • opacity: 1
  • fallbackImage: "blue.png"
  • grainDimension: 1
  • fromColor: "0088c0"
  • toColor: "10a9cf"
  • independentChannels: true
  • s: 0.3
  • n: 3
  • width: 100
  • height: 100
